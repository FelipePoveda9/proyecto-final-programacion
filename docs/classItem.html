Descripción detallada

Clase base para los elementos del juego interactuables.

La clase Item define los elementos básicos de un objeto en el juego, como su posición y sprite asociado. Está pensada para ser una clase base que puede ser heredada por otros objetos más específicos, como HealthBox.
